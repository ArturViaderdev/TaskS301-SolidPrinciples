## English

# O - Open/Closed Principle (OCP)

The exercise defined instruments using a `String` and detected them with an `if` statement.

I modified the code so there's now a common **`Instrument`** interface for all instruments and classes defining each instrument type.

Now if you want to extend the project by adding more instruments, you only need to create classes for each instrument type.

I also separated the `Main` from `InstrumentPlayer`.

L'exercici definia els instruments amb un `String` i els detectava amb un `if`.
